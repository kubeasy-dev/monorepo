- `.inputValidator()` is the correct TanStack Start 1.166.x API for typed POST data (the plan specified `.validator()` which doesn't exist — discovered and fixed immediately during typecheck)

- `getEmailTopicsFn` returns topics with `subscribed: defaultOptIn` at every graceful-degradation branch (no Resend key, no contact ID, or API error)

- The `email_topic` table must be populated (synced from Resend) for topics to appear.
